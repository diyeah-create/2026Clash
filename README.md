# 2026Clash - 全自动最强 Clash 配置生成器

每日自动拉取多个订阅 → 全格式识别（YAML/Base64）→ 严格去重防重名 → 高速测速 → 节点名称增强 → 生成**超级强大配置**（自动选择 + 负载均衡 + 故障转移 + 区域分组 ）+ Loyalsoldier 全网最强规则。

### 功能亮点
- 支持任意数量订阅链接（任意格式）
- 节点名称自动加延迟显示
- 策略组极致丰富：自动选择、低延迟、负载均衡、故障转移、香港/日本/美国/新加坡/台湾/韩国/色情/手动选择
- 规则集每日更新 + 色情专用规则（porn + motv 强制代理）
- 输出 `clash.yaml`（直接导入 Clash 使用）

### 使用方法
1. Fork 本仓库
2. Settings → Secrets and variables → Actions → 新建 `PROXIES_URLS`（每行一个订阅链接）
3. 手动触发 Workflow
4. 下载 `clash.yaml` 即可使用

**推荐 Clash 客户端**：Clash Meta / Clash for Windows / Clash Verge Rev
